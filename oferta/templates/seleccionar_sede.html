Preguntas Frecuentes

{% comment %} FAQ Item 1 {% endcomment %}
¿Cómo funciona el Generador Automático?
Usa un algoritmo de backtracking optimizado que explora todas las combinaciones posibles de secciones, elimina las que tienen conflictos, y las ordena según tus preferencias (horario, ventanas, modalidad virtual, etc.). Te mostramos las 10 mejores opciones con puntuación de 0-100.
{% comment %} FAQ Item 2 {% endcomment %}
¿La oferta académica está actualizada?
Los datos se cargan desde el Excel oficial de tu sede. Recomendamos verificar la información en el sistema oficial de Duoc antes de inscribir, ya que pueden existir cambios de última hora.
{% comment %} FAQ Item 3 {% endcomment %}
¿Puedo guardar múltiples horarios?
Sí, puedes crear y guardar hasta 5 horarios diferentes en tu cuenta. Ideal para comparar opciones o tener un plan B.
